# OVA-IPRED

Proyecto de Objeto Virtual de Aprendizaje (OVA) para la inclusión de material didáctico e informativo acerca de temas como Primeros Auxilios y Procedimientos Opertivos Normalizados (PON)

Este repositorio fue creado para la versión OVA V12 en adelante. El proyecto es realizado bajo la plataforma Adobe Animate.
